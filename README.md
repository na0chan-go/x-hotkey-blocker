# x-hotkey-blocker

Chrome系ブラウザ向けの拡張機能ひな形です。X (x.com / twitter.com) 上で以下ショートカットを押しながらポストを左クリックすると、投稿者のブロックを試行します。

- macOS: `Command + 左クリック`
- Windows/Linux: `Ctrl + 左クリック`

フォロー中ユーザーは、ポストメニュー内の `Unfollow / フォロー解除` 項目を検出した場合にブロック処理をスキップします。

## セットアップ

1. このリポジトリをローカルで開く
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」でこのフォルダを選択

## ファイル構成

- `manifest.json`: Manifest V3設定
- `content.js`: ホットキー + クリック検出とブロック操作

## 注意点

- XのDOM/文言が変わると動作しなくなる可能性があります。
- 誤操作防止やホワイトリストなどは未実装です（今後追加予定）。
